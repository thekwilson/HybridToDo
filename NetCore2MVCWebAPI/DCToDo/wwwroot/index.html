Orionbow ToDo System

This web app has a simple front end UI through javascript to the backend ToDo API that it hosts.
There are two API Controllers that hosts:
  1. /api/values = This controller has some demo api operations AND the Hybrid operations
    • /api/values (get all values)
    • /api/values/id (get by int id)
    • /api/values/devicerecords (get all records over hybrid)
    • /api/values/logsn/SerialNumber (update a record over hybrid)
  2. /api/todo = This controller has demo api operations used by this web page (below)
    • /api/todo (get all todos)
    • /api/todo/todoid(int)
    • /api/todo/todoName(string)

    [Optional Note: you need to set the web app URL in the Javascript variable apiendpoint in this index.html file for the UI below to work]
